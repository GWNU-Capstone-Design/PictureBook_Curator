# PictureBook_Curator
 
## 🖥️ 프로젝트 소개
그림책 이미지를 사진 찍으면 이미지캡셔닝을 통해 그림을 설명하는 키워드를 추출합니다.

GPTs를 활용하여 이미지 내용을 설명하는 시나리오를 생성합니다.

생성한 시나리오를 TTS를 통해 목소리로 들려주는 프로젝트입니다.
